Sobre la app:
*******************************************************************************
El proyecto ‘LudiTest’ consiste en una aplicación móvil para Android que permitiría realizar un quiz de alternativas, con el fin de definir aproximadamente el tipo de personalidad o comportamiento del usuario, 
basado en el modelo DISC, de 4 estilos de personalidad. Cada alternativa está directamente relacionada con uno de esos tipos de personalidad, y para el resultado se considera la mayor predominancia en sus respuestas 
(y en caso de que tenga dos tipos con igual número de respuestas, se le realiza una pregunta extra). 
Una vez terminado el quiz y a partir del resultado, se genera una lista o grid de videojuegos lanzados o por lanzar en el año 2025, y que correspondan con géneros que podrían ser más atractivos para ese tipo de personalidad. 
Adicionalmente, el usuario tendría la opción de guardar cada juego dentro de una lista de favoritos o “juegos por jugar”. Además, las respuestas del usuario quedan guardadas en su perfil.
